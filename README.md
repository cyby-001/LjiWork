# L-记加班

**无广告 · 无弹窗 · 无套路  
一款纯粹为记录「血汗加班费」而生的 Flutter 开源工具

<img width="566" height="566" alt="gaoqiantubiao" src="https://github.com/user-attachments/assets/6d945012-ee6a-4f57-84d5-37fed4b148d0" />

> 应用商店里那些记加班的 App，不是插屏广告就是会员锁功能，看一次气一次。  
> 干脆自己动手，用 免费的Gemini 3 Pro + Flutter 几天肝出来——**L记加班**，彻底告别广告！
> 目前还是比较粗糙的，但是可以满足日常使用了

## 为什么会有人做这个？
- 市面上 99% 的打卡App要么广告满天飞，要么一年会员 98
- 我只想要一个干净、准确、能用 10 年的加班账本
- 所以就有了它

## 核心功能
- 真实农历日历
- 1.5x / 2.0x / 3.0x 加班费精准计算 + 调休智能抵扣
- 周期收入秒出（底薪 + 绩效 + 加班费 – 扣款）
- 年度收入/加班趋势折线图 + 饼图
- 零广告、零联网、零跟踪，数据永远在你手机里
